OCaml-RDF

OCaml-RDF is an OCaml library to manipulate RDF graphs and execute Sparql queries.

Implemented features
  • Three storages available: in memory, in a MySQL or in a Postgresql database,
  • Ability to define your own storages,
  • Transactions,
  • Importing from and exporting to RDF/XML and Turtle formats,
  • Executing Sparql 1.1 queries.
Development

The code is hosted on Github. The development started in late november 2011.

OCaml-RDF is distributed under the GNU LGPL version 3.
