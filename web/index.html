OCaml-RDF

What is OCaml-RDF ?

OCaml-RDF is the OCaml bindings to Redland's Librdf, Raptor 2 and Rasqal. These are libraries to parse, print, load, store, manipulate and query RDF graphs, used in semantic web applications.

OCaml-RDF is an OCaml library to manipulate RDF graphs. It is still in early development stage.

Implemented features:

  • Three storages available: in memory, in a MySQL or in a Postgresql database,
  • Ability to define your own storages,
  • Transactions,
  • Importing from and exporting to RDF/XML format.

OCaml-RDF is hosted on the OCaml forge

The development started in late november 2011.

Installation

The following packages must be installed:

  • OCaml >= 4.0.0 (not tested with prior versions),
  • OCaml-MySQL >= 1.0.4 (not tested with prior versions) for MySQL backend,
  • Postgresql-ocaml >= 2.0.2 (not tested with prior versions) for Postgresql backend,
  • OCamlnet >= 3.6 (not tested with prior versions),
  • Xmlm >= 1.1.0.

Just type 

# ./configure
# make
# make install

This will install the packages rdf, rdf.mysql and rdf.postgresql with ocamlfind.
