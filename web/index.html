OCaml-RDF

What is OCaml-RDF ?

OCaml-RDF are OCaml bindings to Redland's Librdf, Raptor 2 and Rasqal. These are libraries to parse, print, load, store, manipulate and query RDF graphs, used in semantic web applications.

OCaml-RDF is hosted on the OCaml forge

License

OCaml-RDF is distributed under the GNU LGPL version 2.1 or later.

Download

The code is available on Gitorious.

There is no release yet, as the development has just begun.

Installation

Make sure you installed OCaml, librdf, raptor2 and rasqal. On debian distributions, you will need the following packages:
ocaml librdf0-dev libraptor2-dev librasqal3-dev

On other distributions or if you installed these libraries another way, edit the Makefile and adapt the C_INCLUDES variable.

Then just type 

# make
# make install
