Module Rdf_iri

module Rdf_iri: sig .. end
IRIs. See RFC3987.
type iri
IRIs are abstract. Do not compare with generic comparison functions (Pervasives.compare, (=), ...). Use Rdf_iri.equal or Rdf_iri.compare.
exception Invalid_iri of string * string
When the IRI is parsed, Invalid_iri (string, message) is raised if the given string is invalid, with an additional message.
val string : iri -> string
Create a string from an IRI.
val iri : ?check:bool -> string -> iri
Create an IRI from a string.
Raises Invalid_iri in case the string does not represent a valid IRI.
check : can be specified to false to prevent IRI parsing. Parsing will be done later if needed, and may then raise Invalid_iri.
val ensure_absolute : iri -> string -> iri
ensure_absolute base s creates an IRI from s, using the given base if s is relative. Apply these rules.
val append : ?check:bool -> iri -> string -> iri
Append the given string to the given IRI.
check : can be specified to false to prevent IRI parsing.
val concat : iri -> string -> iri
Add the given string to the path of the given IRI.
val to_uri : iri -> Rdf_uri.uri
to_uri iri encodes the given IRI and tries to create an URI. May raise Rdf_uri.Invalid_uri is the IRI cannot be mapped to an URI.
val of_uri : Rdf_uri.uri -> iri
of_uri uri get the string representation of the given uri, then percent-decode it and build an iri from this decoded string, using Rdf_iri.iri.
