Installation
With opam

OCaml-RDF is available in OPAM: 

$ opam install rdf
Manual installation
Dependencies

Compiling OCaml-RDF requires the following software to be installed:

  • OCaml >= 4.00,
  • Findlib >= 1.3.3 (should work with priori versions),
  • OCamlnet >= 3.6.5,
  • OCaml-MySQL >= 1.1.1 (not tested with prior versions) for MySQL backend,
  • Postgresql-ocaml >= 2.0.2 (not tested with prior versions) for Postgresql backend,
  • Xmlm >= 1.1.1,
  • Uuidm >= 0.9.5,
  • ulex >= 1.1 (not tested with prior versions),
  • Menhir >= 20120123 (not tested with prior versions),
  • Cryptokit >= 1.7 (not tested with prior versions),
  • PCRE-OCaml >= 7.0.2 (not tested with prior versions),
  • [optional] Config-file >= 1.1 to run the test suite,
Compiling and installing

After extracting the source tree, go to the extracted directory and type: 

$ ./configure
$ make all
$ make install # with sufficient rights

This will install the packages rdf, rdf.mysql and rdf.postgresql with ocamlfind.
